# laravel-neverbounce

A laravel wrapper for the NeverBounceAPI-PHP.

Allows up to 1,000 email verifications per month.

https://developers.neverbounce.com/docs
https://github.com/NeverBounce/NeverBounceAPI-PHP

Supported Scenario:
